## Strength Levels

| Entropy | Level | Security |
|---------|-------|----------|
| < 28 bits | Weak | Can be cracked quickly |
| 28-50 bits | Fair | Moderate security |
| 50-80 bits | Good | Strong security |
| 80+ bits | Excellent | Very strong security |
